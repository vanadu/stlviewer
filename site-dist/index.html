whitty-js Overview

witty-js is designed to help HTML Email developers create fully responsive HTML emails using a tabular layout in situations where fluid layouts aren't appropriate, for example when the document will undergo regulatory or legal reviews in which total control of word wrap and line breaks are required, or when the document has to reflect a specific branding or layout in which vertical positioning of elements without content resizing is critical.

Witty helps developers keep track of image dimension property values so they can be reused in multiple places in the document. In conjunction with a clipboard manager, developers can quickly build a store of all the HTML and CSS code snippets relevant for a specific image and insert them at various positions in the document with a few mouseclicks.
